## Les modules du cœur

* `require("…/fichier.js")` = perso
* `require("module")` = core ou npm
* cf. [nodejs.org/en/docs/](https://nodejs.org/dist/latest-v6.x/docs/api/)

### Les principaux

* Fichiers : `fs`, `path`
* Réseaux : `http`, `net`
* Système : `os`, `child_process`
* Utilitaires : `util`
